# What does it do?
It adds, *attempts to add*, Final Fantasy themed monsters, items, professions and more to Cataclysm: Brighter Nights.

## Currently Adds:
### Monsters
- **Chocobo**. A large yellow bird, can be tamed with `bird food`, and ridden like a horse with Survival Skill 3 and `Horse tack`. They can be found naturally in forests.
- **Chocobo Chick**. A baby chocobo bird. grows into a chocobo.
### Items
- **Chocobo Egg**. Laid by adult chocobos, can hatch a chocobo chick when expired.

---
# How do I Install it?
1. Clone/Download/Copy the repo and move the `FinalFantasyMod` folder to your `Cataclysm-Brighter-Nights/Mods/` directory, so it reads `Cataclysm-Brighter-Nights/Mods/FinalFantasyMod`.

2. Create a new world, and select FFantasyMod on the first step of world creation.
